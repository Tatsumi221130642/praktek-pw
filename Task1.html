Web Design

Web design encompasses many diffrent skill and diciplins in the production and maintence of website. The diffrent areas of web design include web grapich design; user interface design(UI design); authoring, including standardised code and proprietary software; user expirience design(UX design); and search engine optimization. Often many individual will work cover them all. The term " web design " is normally used to describe the design process relating to the front-end (client side) design of a website including writing markup. Web design partially overlaps web enginering in the broader scope of web development. Web designers are expected to have awareness of usability and be up to date with web accessbility suidelines
